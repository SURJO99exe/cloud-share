# Cloud Share - Premium File Sharing App

## বাংলা

### পরিচিতি
Cloud Share একটি প্রিমিয়াম ফাইল শেয়ারিং অ্যাপ্লিকেশন যা আপনাকে সহজে ফাইল আপলোড, শেয়ার এবং ম্যানেজ করতে সাহায্য করে।

### বৈশিষ্ট্য
- 🚀 দ্রুত ফাইল আপলোড এবং ডাউনলোড
- 📱 মোবাইল এবং ডেস্কটোপ সাপোর্ট
- 🔒 নিরাপদ ফাইল শেয়ারিং
- 🌙 ডার্ক মোড সাপোর্ট
- 📋 ক্লিপবোর্ড থেকে পেস্ট করে আপলোড
- 🗂️ ফাইল ক্যাটাগরি অনুযায়ী ফিল্টার
- 📊 স্টোরেজ স্ট্যাটস দেখুন
- 🌐 সার্ভার অন/অফ ফাংশন

### ইনস্টলেশন

#### ১. Android APK তৈরি করুন
```bash
npm install
npx cap add android
npx cap sync android
cd android
./gradlew assembleRelease
```

#### ২. Desktop App তৈরি করুন
```bash
npm install
npm run electron-build
```

#### ৩. সরাসরি চালান
```bash
npm start
```

### ব্যবহার
1. অ্যাপ চালু করুন
2. ফাইল ড্র্যাগ করে আনুন অথবা "Upload" বাটনে ক্লিক করুন
3. Ctrl+V চেপে ক্লিপবোর্ড থেকে পেস্ট করুন
4. শেয়ার লিঙ্ক কপি করুন
5. সার্ভার অন/অফ করুন

## English

### Introduction
Cloud Share is a premium file sharing application that helps you easily upload, share, and manage files.

### Features
- 🚀 Fast file upload and download
- 📱 Mobile and Desktop support
- 🔒 Secure file sharing
- 🌙 Dark mode support
- 📋 Paste upload from clipboard
- 🗂️ Filter files by category
- 📊 View storage stats
- 🌐 Server on/off function

### Installation

#### 1. Build Android APK
```bash
npm install
npx cap add android
npx cap sync android
cd android
./gradlew assembleRelease
```

#### 2. Build Desktop App
```bash
npm install
npm run electron-build
```

#### 3. Run Directly
```bash
npm start
```

### Usage
1. Start the app
2. Drag files or click "Upload" button
3. Press Ctrl+V to paste from clipboard
4. Copy share links
5. Toggle server on/off
